## Homebrew installation

```bash
$ ruby -e "$(curl -fsSL https://raw.github.com/Homebrew/homebrew/go/install)"
$ brew update
```

## Install commons

```bash
$ brew install openssl
$ brew install wget
$ brew install curl tree git
$ brew install vim
$ brew install imagemagick
```


## Configure PATH

```bash
$ echo 'export PATH="/usr/local/bin:$PATH"' >> ~/.zlogin
$ source ~/.zlogin
```

```bash
$ brew doctor
```
