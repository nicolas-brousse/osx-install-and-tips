# Homebrew

```bash
$ cd ~
$ mkdir .homebrew && curl -L https://github.com/Homebrew/homebrew/tarball/master | tar xz --strip 1 -C .homebrew
$ ~/.homebrew/bin/brew update

$ brew install openssl
$ brew install wget
$ brew install curl tree git (git-core ?)
$ brew install vim
$ brew install pow
```
